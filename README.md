# 个人日记本

一个功能丰富的日记本应用，支持多用户系统和 Flask Web 界面。

## 功能特性

### 用户系统
- 🔐 **用户注册和登录** - 支持用户名密码认证
- 👤 **多用户支持** - 每位用户拥有独立的日记空间
- 🛡️ **权限管理** - 普通用户和管理员两种角色
- 📛 **管理员面板** - 管理员可管理所有用户（创建、删除、禁用、分配角色）
- 🔒 **登录安全** - 登录失败次数限制（5次失败锁定5分钟）
- ⏱️ **会话超时** - 30分钟无活动自动登出

### 日记功能
- ✍️ 写日记（支持今天、相对日期或指定日期）
- 📖 阅读日记（Markdown 渲染）
- ✏️ 编辑日记（保留原有标签，支持修改内容）
- 🗑️ 删除日记（带确认页面，防止误删）
- 🔍 搜索日记内容（支持多关键词）
- 📋 列出所有日记

### 标签与心情
- 🏷️ 标签系统（支持按标签分类，带 XSS 防护）
- 😊 心情记录（记录每日心情状态）

### 图片上传
- 📷 图片上传功能（支持 PNG、JPG、JPEG、GIF、WebP）
- 🔒 安全验证（文件扩展名 + Magic Number 双重检查）
- 🆔 唯一文件名（使用 UUID 避免文件名冲突）

### 导入导出与备份
- 📤 导出功能（导出为 JSON、CSV、Markdown、ZIP 格式）
- 📥 导入功能（从 JSON/ZIP 文件导入日记）
- 💾 数据库持久化存储

### 界面与通知
- 🌐 Flask Web 界面（Bootstrap 响应式设计）
- 🔔 通知功能（Toast 弹窗提示，不占用页面布局）
- ⚙️ 设置管理（主题、备份等配置）

## 安装依赖

```bash
pip install -r requirements.txt
```

### 依赖项说明

- **Flask>=2.3.0** - Web 应用框架
- **SQLAlchemy** - 数据库 ORM
- **markdown==3.10.2** - Markdown 渲染
- **colorama==0.4.4** - 终端彩色输出
- **python-dotenv** - 环境变量支持

- **写日记** - 创建新日记，支持 Markdown 格式和标签

- **统计页面** - 查看日记统计数据和标签使用频率

## 用户角色说明

| 角色 | 权限 |
|------|------|
| 普通用户 | 创建、查看、编辑、删除自己的日记 |
| 管理员 | 所有普通用户权限 + 管理所有用户账户 |

## 相对日期支持

在输入日期时，可以使用以下相对日期：
- `今天` - 今天的日期
- `昨天` - 昨天的日期
- `前天` - 前天的日期

## Markdown 支持

日记内容支持 Markdown 格式：

| 语法 | 显示效果 |
|------|----------|
| `# 标题` | 标题 |
| `**粗体**` | **粗体** |
| `*斜体*` | *斜体* |
| `- 列表项` | 列表 |
| `> 引用` | 引用 |

## 数据存储

- **SQLite 数据库** - 用户、日记、标签、心情的持久化存储
- **登录追踪** - 数据库表记录登录尝试（防暴力破解）
- **图片存储** - 本地文件系统，UUID 唯一命名

## 安全特性

- 🛡️ **密码哈希** - PBKDF2 + SHA256，210000 次迭代
- 🔒 **登录锁定** - 5次失败锁定5分钟
- ⏱️ **会话超时** - 30分钟无活动自动登出
- 📷 **文件上传安全** - 扩展名验证 + Magic Number 检查
- 🏷️ **输入验证** - 标签和内容 XSS 防护转义
- 🆔 **文件名安全** - UUID 替代时间戳，避免路径遍历

## 数据备份

- 支持导出为 JSON、CSV、Markdown、ZIP 格式
- 备份文件可下载保存
