# 个人日记本

一个功能丰富的日记本应用，支持多用户系统和 Flask Web 界面。

## 功能特性

### 用户系统
- 🔐 **用户注册和登录** - 支持用户名密码认证
- 👤 **多用户支持** - 每位用户拥有独立的日记空间
- 🛡️ **权限管理** - 普通用户和管理员两种角色
- 📛 **管理员面板** - 管理员可管理所有用户（创建、删除、禁用、分配角色）

### 日记功能
- ✍️ 写日记（支持今天、相对日期或指定日期）
- 📖 阅读日记（Markdown 渲染 + 彩色输出）
- ✏️ 编辑日记（保留原有标签，支持修改内容）
- 🗑️ 删除日记
- 🔍 搜索日记内容（支持多关键词、模糊搜索）
- 📋 列出所有日记

### 标签与心情
- 🏷️ 标签系统（支持按标签分类）
- 😊 心情记录（记录每日心情状态）

### 导入导出与备份
- 📤 导出功能（导出为 JSON、CSV、Markdown、ZIP 格式）
- 📥 导入功能（从 JSON/ZIP 文件导入日记）
- 💾 退出时自动备份

### 界面与通知
- 🌐 Flask Web 界面（Bootstrap 响应式设计）
- 🔔 通知功能（Toast 弹窗提示，不占用页面布局）
- ⚙️ 设置管理（主题、备份、自动保存等配置）

## 安装依赖

```bash
pip install -r requirements.txt
```

### 依赖项说明

- **Flask==2.0.1** - Web 应用框架
- **SQLAlchemy** - 数据库 ORM
- **markdown==3.3.4** - Markdown 渲染
- **colorama==0.4.4** - 终端彩色输出

## 快速开始

### 1. 启动应用

```bash
python app.py
```

然后在浏览器中访问：http://127.0.0.1:5000

### 2. 登录账户

- **管理员账号**：admin / admin123
- **普通用户**：请先注册账号

### 3. 开始使用

- 点击「写日记」创建新日记
- 点击日记日期查看历史日记
- 使用标签和搜索功能整理和查找日记

## Web 界面功能

- **首页** - 显示日历视图、标签云和统计数据
- **写日记** - 创建新日记，支持 Markdown 格式和标签
- **查看日记** - 阅读日记内容，支持 Markdown 渲染
- **编辑日记** - 修改已有日记内容和标签
- **搜索日记** - 按关键词搜索日记，显示匹配结果
- **标签浏览** - 按标签分类浏览日记
- **统计页面** - 查看日记统计数据和标签使用频率
- **导入导出** - 导入或导出日记（JSON/CSV/Markdown/ZIP）
- **设置页面** - 管理应用配置（主题、备份、通知等）
- **管理员面板** - 管理用户账户和权限

## 用户角色说明

| 角色 | 权限 |
|------|------|
| 普通用户 | 创建、查看、编辑、删除自己的日记 |
| 管理员 | 所有普通用户权限 + 管理所有用户账户 |

## 相对日期支持

在输入日期时，可以使用以下相对日期：
- `今天` - 今天的日期
- `昨天` - 昨天的日期
- `前天` - 前天的日期
- `3天前` / `N天前` - N 天前的日期
- `上周` - 上周的日期
- `上个月` - 上个月的日期

## Markdown 支持

日记内容支持 Markdown 格式，读取时以彩色终端输出：

| 语法 | 显示效果 |
|------|----------|
| `# 标题` | 绿色高亮标题 |
| `## 二级标题` | 绿色 |
| `### 三级标题` | 青色 |
| `**粗体**` | 黄色高亮 |
| `*斜体*` | 紫红色 |
| `- 列表项` | 带黄色圆点 |
| `> 引用` | 蓝色带竖线 |

## 数据备份

- 退出时会自动备份 `entries` 文件夹
- 备份文件格式：`entries_backup_年月日_时分秒`
- 支持备份管理：列出、恢复、删除备份
- 支持导出为 JSON、CSV、Markdown、ZIP 格式

## 统计功能

- 日记总数
- 总字数统计
- 总字符数统计
- 标签使用频率排行
