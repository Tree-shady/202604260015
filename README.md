# 个人日记本

一个功能丰富的日记本应用，支持多用户系统和 Flask Web 界面。

## 功能特性

### 用户系统
- 🔐 **用户注册和登录** - 支持用户名密码认证
- 👤 **多用户支持** - 每位用户拥有独立的日记空间
- 🛡️ **权限管理** - 普通用户和管理员两种角色
- 📛 **管理员面板** - 管理员可管理所有用户（创建、删除、禁用、分配角色）
- 🔒 **登录安全** - 登录失败次数限制（5次失败锁定5分钟）
- ⏱️ **会话超时** - 30分钟无活动自动登出

### 日记功能
- ✍️ **写日记**（支持今天、相对日期或指定日期）
- 📖 **阅读日记**（Markdown 渲染）
- ✏️ **编辑日记**（保留原有标签，支持修改内容）
- 🗑️ **删除日记**（带确认页面，防止误删）
- 🔍 **搜索日记**（支持多关键词、日期范围、心情、标签筛选）
- 📋 **列出所有日记**
- 📝 **日记模板**（8种预设模板：日常、工作、学习、旅行、健康、感恩、梦境、项目）
- 🔔 **日记提醒**（可设置定时提醒，帮助养成写日记习惯）

### 富文本编辑器
- 🎨 **Quill 编辑器** - 现代化富文本编辑体验
- 📝 **格式工具** - 标题、粗体、斜体、下划线、删除线
- 📋 **列表支持** - 有序列表、无序列表
- 💬 **引用和代码** - 块引用、代码块
- 🔗 **链接图片** - 支持插入链接和图片
- 🔢 **字数统计** - 实时显示字符数
- ⌨️ **快捷键支持**：
  - `Ctrl+S` - 保存日记
  - `Ctrl+N` - 新建日记
  - `Ctrl+B` - 加粗
  - `Ctrl+I` - 斜体

### 标签与心情
- 🏷️ **标签系统**（支持按标签分类，带 XSS 防护）
- 😊 **心情记录**（8种心情状态：开心、兴奋、平静、疲惫、难过、生气、焦虑、一般）
- 📊 **心情趋势图表**（30天心情折线图、心情分布饼图）

### 图片上传
- 📷 **图片上传功能**（支持 PNG、JPG、JPEG、GIF、WebP）
- 🔒 **安全验证**（文件扩展名 + Magic Number 双重检查）
- 🆔 **唯一文件名**（使用 UUID 避免文件名冲突）

### 高级搜索
- 🔍 **关键词搜索** - 搜索日记内容
- 📅 **日期范围筛选** - 按开始和结束日期筛选
- 😊 **心情筛选** - 按心情类型多选筛选
- 🏷️ **标签筛选** - 按标签分类筛选
- ✨ **结果高亮** - 搜索关键词高亮显示

### 数据可视化统计
- 📊 **统计面板** - 日记总数、总字数、标签总数
- 📈 **心情分布图** - 饼图展示各心情类型占比
- 📅 **月度柱状图** - 显示每月日记数量
- 📉 **心情趋势图** - 30天心情变化折线图
- 🏷️ **标签排行** - 横向柱状图显示热门标签 Top 10

### 导入导出与备份
- 📤 **导出功能**（导出为 JSON、CSV、Markdown、ZIP 格式）
- 📥 **导入功能**（从 JSON/ZIP 文件导入日记）
- 💾 **数据库持久化存储**

### 界面与通知
- 🌐 **Flask Web 界面**（Bootstrap 5 响应式设计）
- 📱 **现代化 UI** - 侧边栏导航、卡片式布局
- 🔔 **通知功能**（Toast 弹窗提示，不占用页面布局）
- ⚙️ **设置管理**（主题、备份等配置）
- 👋 **问候语功能**（启动时显示问候语，支持本地/在线API获取每日问候）

### 连续打卡系统
- 🔥 **连续打卡统计** - 实时记录连续写日记天数
- 🏆 **成就徽章** - 达到特定天数获得对应徽章（1天🌱、3天⭐、7天🌟、14天💫、30天🏆、50天👑、100天💎）
- 📊 **历史记录** - 统计最长连续天数和总日记数
- 🎯 **激励消息** - 根据连续天数显示鼓励语

### 年度回顾报告
- 📅 **年度统计** - 日记数量、总字数、平均字数统计
- 🏷️ **热门标签** - 年度使用最频繁的标签
- 😊 **心情分布** - 全年心情统计分析
- 📈 **月度分布** - 每月日记数量图表
- 🏆 **年度成就** - 年度关键词总结

## 数据存储

- **SQLite 数据库** - 用户、日记、标签、心情的持久化存储
- **登录追踪** - 数据库表记录登录尝试（防暴力破解）
- **图片存储** - 本地文件系统，UUID 唯一命名

## 安全特性

- 🛡️ **密码哈希** - PBKDF2 + SHA256，210000 次迭代
- 🔒 **登录锁定** - 5次失败锁定5分钟
- ⏱️ **会话超时** - 30分钟无活动自动登出
- 📷 **文件上传安全** - 扩展名验证 + Magic Number 检查
- 🏷️ **输入验证** - 标签和内容 XSS 防护转义
- 🆔 **文件名安全** - UUID 替代时间戳，避免路径遍历

## 数据备份

- 支持导出为 JSON、CSV、Markdown、ZIP 格式
- 备份文件可下载保存

## 性能优化

- 🚀 **数据缓存** - 使用 Flask-Caching 对频繁访问的数据进行缓存
- ⚡ **数据库查询优化** - 使用 SQL 聚合查询替代 Python 循环统计
- 💾 **连接池管理** - 自动管理数据库连接，提高并发性能

## 日记提醒

设置每日提醒，帮助养成写日记的好习惯：

1. 进入「提醒设置」页面
2. 开启提醒功能
3. 设置希望的提醒时间（默认21:00）
4. 保存设置

系统会在设定时间提醒您写日记（需要浏览器允许通知权限）。

## 技术栈

- **后端**: Flask 3.0 + SQLAlchemy 2.0
- **前端**: Bootstrap 5 + Quill Editor + Chart.js
- **数据库**: SQLite
- **缓存**: Flask-Caching (SimpleCache)
