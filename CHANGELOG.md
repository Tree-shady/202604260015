## v1.2.0 (2026-05-15)
**Build #5**

### ✨ 新增功能
- 配置验证系统 - 加载和保存配置时自动验证和修复
- 安全响应头中间件 - 添加 CSP、X-Content-Type-Options、X-Frame-Options 等安全头

### ⚡ 功能优化
- 移除硬编码密码 - 首次运行时自动生成安全的随机管理员密码
- 数据库查询优化 - 使用 SQL 聚合替代 Python 循环（标签统计、月度统计、心情统计）
- Web 页面布局优化 - 主内容区域最大宽度 1400px，居中显示
- 统计卡片优化 - 最小宽度从 250px 调整为 200px，内边距优化
- 首页卡片间距优化 - 从 mb-4 调整为 mb-3，更加紧凑
- 字体大小和标题层级优化 - 使页面视觉更加协调
- 心情分布卡片优化 - 间距和内边距调整，最小宽度 70px
- README 文档全面更新 - 新增本次优化的详细说明

### 🛡️ 安全增强
- 添加安全响应头 - X-Content-Type-Options、X-Frame-Options、X-XSS-Protection、CSP、Referrer-Policy
- 安全密码生成 - 使用 secrets.token_urlsafe(16) 生成 16 字符安全密码
- 配置验证 - 防止错误配置导致的安全问题
