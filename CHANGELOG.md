## 📝 更新详情

### v1.1.0 (2026-05-07)
**Build #2** · **类型**: Minor Release · **状态**: 稳定

#### ✨ 新增功能

| 功能 | 描述 | 影响范围 |
|------|------|----------|
| **深色主题** | 支持一键切换明暗主题，自动保存用户偏好到 localStorage | 前端界面 |
| **AI写作提示** | 内置写作灵感提示系统，支持5种分类（日常/反思/创意/感恩/季节），支持按心情和时间智能推荐 | 写日记页面 |
| **收藏功能** | 支持收藏重要日记，方便快速访问，使用 JSON 文件持久化存储 | 日记管理 |
| **密码强度检测** | 实时显示密码强度进度条，包含大写/小写/数字要求验证 | 用户注册/修改密码 |
| **请求限流** | API接口限流保护，防止恶意请求和暴力破解 | 后端API |

**技术实现细节**：
- 深色主题使用 CSS 变量实现，支持平滑切换，包含完整的颜色主题
- 写作提示支持实时获取和插入到编辑器
- 收藏功能支持添加、移除、查询操作

#### 🛡️ 安全增强

| 修复项 | 描述 | 风险等级 |
|--------|------|----------|
| **CSRF保护** | 为所有POST/DELETE API添加CSRF令牌验证，防止跨站请求伪造 | 🔴 高 |
| **日志规范** | 将print语句替换为logging模块，便于生产环境日志管理 | 🟡 中 |
| **未使用导入清理** | 移除rate_limit.py中未使用的flask模块（flash, redirect, url_for） | 🟢 低 |
| **密码验证统一** | 前后端密码规则同步为8字符+大小写字母+数字，确保数据一致性 | 🔴 高 |

#### 🐛 Bug修复

| 问题 | 描述 | 修复方式 |
|------|------|----------|
| 密码特殊字符解析 | 密码中包含`@`符号会被错误解析为端口号 | 使用urllib.parse.quote_plus进行URL编码 |
| 远程配置丢失 | 数据库配置重启后丢失 | 修改保存逻辑，支持部分更新，未填写字段保留原值 |
| 局域网访问 | 只能通过127.0.0.1访问 | 修改绑定地址为0.0.0.0，支持局域网访问 |

#### ⚡ 功能优化

| 优化项 | 描述 | 效果 |
|--------|------|------|
| **字体优化** | 调整页面字体大小为适中尺寸（15px），增加行高 | 提升阅读体验 |
| **README更新** | 全面更新项目文档，添加新功能说明和使用指南 | 提高文档质量 |
| **更新日志系统** | 新增自动记录更新日志功能，支持版本号和构建号管理 | 便于版本追踪和问题排查 |

### v1.0.0 (2026-05-01)
**Build #1** · **类型**: Initial Release · **状态**: 稳定

#### 🚀 初始功能

**用户系统**
- 🔐 用户注册和登录（支持用户名密码认证）
- 👤 多用户支持（每位用户拥有独立的日记空间）
- 🛡️ 权限管理（普通用户和管理员两种角色）
- 📛 管理员面板（管理所有用户：创建、删除、禁用、分配角色）
- 🔒 登录安全（5次失败锁定30分钟）
- ⏱️ 会话超时（30分钟无活动自动登出）

**日记功能**
- ✍️ 写日记（支持今天、相对日期或指定日期）
- 📖 阅读日记（Markdown 渲染）
- ✏️ 编辑日记（保留原有标签，支持修改内容）
- 🗑️ 删除日记（带确认页面，防止误删）
- 🔍 搜索日记（支持多关键词、日期范围、心情、标签筛选）
- 📋 列出所有日记
- 📝 日记模板（8种预设模板：日常、工作、学习、旅行、健康、感恩、梦境、项目）
- 🔔 日记提醒（可设置定时提醒，帮助养成写日记习惯）

**富文本编辑器**
- 🎨 Quill 编辑器（现代化富文本编辑体验）
- 📝 格式工具（标题、粗体、斜体、下划线、删除线）
- 📋 列表支持（有序列表、无序列表）
- 💬 引用和代码（块引用、代码块）
- 🔗 链接图片（支持插入链接和图片）
- 🔢 字数统计（实时显示字符数）
- ⌨️ 快捷键支持（Ctrl+S保存、Ctrl+N新建、Ctrl+B加粗、Ctrl+I斜体）

**标签与心情**
- 🏷️ 标签系统（支持按标签分类，带 XSS 防护）
- 😊 心情记录（8种心情状态：开心、兴奋、平静、疲惫、难过、生气、焦虑、一般）
- 📊 心情趋势图表（30天心情折线图、心情分布饼图）

**数据可视化统计**
- 📊 统计面板（日记总数、总字数、标签总数）
- 📈 心情分布图（饼图展示各心情类型占比）
- 📅 月度柱状图（显示每月日记数量）
- 📉 心情趋势图（30天心情变化折线图）
- 🏷️ 标签排行（横向柱状图显示热门标签 Top 10）

**导入导出与备份**
- 📤 导出功能（导出为 JSON、CSV、Markdown、ZIP 格式）
- 📥 导入功能（从 JSON/ZIP 文件导入日记）
- 💾 数据库持久化存储

**连续打卡系统**
- 🔥 连续打卡统计（实时记录连续写日记天数）
- 🏆 成就徽章（1天🌱、3天⭐、7天🌟、14天💫、30天🏆、50天👑、100天💎）
- 📊 历史记录（统计最长连续天数和总日记数）
- 🎯 激励消息（根据连续天数显示鼓励语）

**年度回顾报告**
- 📅 年度统计（日记数量、总字数、平均字数统计）
- 🏷️ 热门标签（年度使用最频繁的标签）
- 😊 心情分布（全年心情统计分析）
- 📈 月度分布（每月日记数量图表）
- 🏆 年度成就（年度关键词总结）

**写作习惯分析**
- 📊 热力图（年度写作热力图展示）
- ⏰ 最佳时间分析（分析最常写作的小时和星期）
- 📈 坚持率计算（自动计算日记完成率）
- 🔥 连续记录（当前和最长连续记录统计）

**写作挑战系统**
- 🏆 多种挑战（7天、30天、100天、365天挑战）
- 📍 进度追踪（实时显示挑战完成进度）
- 🎖️ 成就徽章（完成挑战获得对应徽章）
- ⭐ 状态管理（进行中/已完成/可开始）

**回忆功能**
- 🎞️ 那年今日（查看去年/多年前的今天写的日记）
- 🏅 里程碑追踪（第1篇、第10篇、第100篇等里程碑日记）
- 📝 日记统计（总日记数量和第一篇日记日期）

**远程数据库支持**
- 首次登录/注册时可选择存储位置
- 支持 PostgreSQL 和 MySQL 数据库
- 设置页面可修改存储位置和配置远程数据库
- 支持环境变量配置敏感信息

**图片上传**
- 📷 图片上传功能（支持 PNG、JPG、JPEG、GIF、WebP）
- 🔒 安全验证（文件扩展名 + Magic Number 双重检查）
- 🆔 唯一文件名（使用 UUID 避免文件名冲突）

**安全特性**
- 🛡️ 密码哈希（PBKDF2 + SHA256，210000 次迭代）
- 🔒 登录锁定（5次失败锁定30分钟）
- ⏱️ 会话超时（30分钟无活动自动登出）
- 📷 文件上传安全（扩展名验证 + Magic Number 检查）
- 🏷️ 输入验证（标签和内容 XSS 防护转义）
- 🆔 文件名安全（UUID 替代时间戳，避免路径遍历）

**界面与主题**
- 🌐 Flask Web 界面（Bootstrap 5 响应式设计）
- 📱 现代化 UI（侧边栏导航、卡片式布局）
- 🔔 通知功能（Toast 弹窗提示，不占用页面布局）
- ⚙️ 设置管理（主题、备份等配置）
- 👋 问候语功能（启动时显示问候语，支持本地/在线API获取每日问候）

**桌面图形化应用（PyQt6）**
- 🖥️ 原生桌面体验（支持本地运行，无需浏览器）
- 👤 用户认证（支持登录、注册）
- 🔧 用户管理（管理员可以创建、删除用户）
- 📁 数据迁移（从旧版文件系统迁移到数据库）
- 📋 本地备份（完整的备份、恢复功能）
- 🎨 主题支持（支持明暗主题切换）
- ☁️ 云端存储（支持本地/云端数据库切换）
